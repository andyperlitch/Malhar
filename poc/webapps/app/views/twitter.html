Twitter Top URLs Open in Console

This demo application proceeds short text messages (tweets) and determines most frequently cited URLs during the last 5 sec. The results are calculated over the sliding window 5 sec and continuously updated each 0.5 sec. For demo purposes, we use the combination of real and auto-generated data. The application is totally unaware of the fact, that it uses developer version of Twitter “firehose”, which provides 1% sample of all tweets happening, and each tweet is cloned around 100 factor, before it actually reaches the app.
