Mobile Geo-Location Monitoring Open in Console

This demo application highlights the ability to register specific mobile numbers and start receiving geo-location updates in real-time. While the events arrive at high volume of 50k to 250k updates per second, but the application only pushes out the updates for the registered phones, making it possible to visualize using simple web user interface. For privacy concerns, the data for this demo is automatically generated.
