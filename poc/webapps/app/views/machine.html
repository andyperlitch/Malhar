Large Scale Monitoring and Analysis Open in Console

This demo application constantly monitors samples of CPU, RAM and HD utilization coming from a heterogeneous installed base of multiple devices located at multiple data centers and used by different customers. The application receives sensor events from each device including the device ID, device model, the OS version, other installed application versions, and customer attributes. It tracks the status of the devices across every combination of dimensions of the events.. This allows both immediate analysis monitoring in real time of anomalies - e.g. notify me when average CPU utilization exceeds 70% for particular combination of model, customer and software. While influenced by real customers, because of security concerns, this demo uses randomly generated data.
