Large Scale Monitoring and Analysis Open in Console

This demo application constantly monitors samples of CPU, RAM and HD utilization coming from heterogeneous installed base of multiple devices located at multiple data centers and used by different customers. The application constantly generates and calculates averages for multiple analytic dimensions, including product type, OS, several software versions and customer and stores the data into fast storage (in this case Redis). This allows both immediate analysis by any combination of analysis dimensions and also allows to generate alarms in real time (e.g. notify me when average CPU utilization exceeds 70% for particular combination of model, customer and software). While influenced by real customer, because of security concerns, this demo uses randomly generated data.
