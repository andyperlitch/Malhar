Analyze an Apache Log

Interested in seeing a little bit of the power of the DataTorrent platform? This page will help you set up an application that will ingest an Apache Server Access Log.

Location of Log File

First, you will need to specify a location of a log file. It should be located on the same machine that the DataTorrent Gateway is running on (where this page is served from).

This should be an absolute path, eg. /var/log/apache2/access.log.
