Build a Real-Time ETL Application

The Extract, Transform, Load (ETL) pattern is an approach to solving a very common problem: ingesting data, transforming and filtering that data, and finally storing it somewhere, perhaps in a database or a file system.

Constructing an ETL application with the DataTorrent platform provides several perks for dev and ops teams alike: real-time insights, high availability, fault-tolerance, and scalable ingestion, to name a few. Use this page to construct and customize your own ETL application.
